Importing Stylesheets in a React module [Ordered: Alphabetically]
Import Description
import cssFilePath:str - imports a css file into this React module
- if the file has (regular) .css extension then the file's styles are available in all Components (child, parent... everyhere) and in the original HTML document
- if the file has .module.css extension IDs and Classes

- other document element stylings (like div, body, etc...) are still available in the whole HTML document

Using these isolated IDs and Classes in JSX

<element className={cssModule.class}><element />

<element id={cssModule.id}><element />

are available only in this module

Inline Styling in JSX [Ordered: Alphabetically]
JSX Description
<element style={obj} <element> - inline styles are passed in a JavaScript object inside the JSX expression (ex: <p style={{color:blue}}>)
- double CSS property names are camelCased (ex: backgroundColor)

Referencing CSS IDs and Classes from JSX [Ordered: Alphabetically]
JSX Description
<element className=className><element> - CSS classes in stylesheets (or document head styles) are referenced as className instead of class
<element id=id><element> - CSS IDs in stylesheets (or document head styles) are referenced as id (just like in normal HTML)

Importing stylesheets in a React module

DEMO 
    import React from 'react';
    import ReactDOM from 'react-dom';
    
    import myStyle from './myStyle.css';                                               // imported stylesheet styles are available everywhere (just like we would reference the 'myStyle.css'. stylesheet from the document's head)    
    import myStyle2 from './myStyle.module.css';                                       // imported stylesheet ID and Class styles are available only in this module through the 'myStyle2' module reference   
    
    class TestComponent extends React.Component{
        render(){
            let style_1 = {color:'red', backgroundColor:'purple', textDecoration:'underline'}
            return(
                <div>
                    <p className="myClass"> Element - 1 </p>                           // style class 'myClass' from './myStyle.css' (available in all React Modules)     
                    <p id="myId"> Element - 2 </p>                                     // style id 'myId' from './myStyle.css' (available in all React Modules)    
                    <p className={myStyle2.testClass}> Element - 3 </p>                // -! style class 'testClass' from './myStyle.module.css' (available only in this module)     
                    <p id={myStyle2.testId}> Element - 4 </p>                          // -! style id 'testId' from './myStyle.module.css' (available only in this module)     
                </div>
            )
        }
    }
    
    ReactDOM.render(<TestComponent />, document.getElementById('testElement'));

// React main script file (index.js) --------------------------------------------------
// ------------------------------------------------------------------------------------
    import React from 'react';
    import ReactDOM from 'react-dom';
    
    import ChildComponent from './child1.js';
    import ChildComponent2 from './child2.js';
    
    class TestComponent extends React.Component{
        render(){
            let style_1 = {color:'red', backgroundColor:'purple', textDecoration:'underline'}
            return(
                <div>
                    <p className="myClass"> Element - 1 </p>                           // 'myClass' from './myStyle.module.css' is not available here or ensewhere    
                    <p id="myId"> Element - 2 </p>                                     // 'myId' from './myStyle.module.css' is not available here or elsewhere 
                    <ChildComponent />
                    <ChildComponent2 />
                </div>
            )
        }
    }
    
    ReactDOM.render(, document.getElementById('testElement'));
    
// Child Component file (child1.js) ---------------------------------------------------
// ------------------------------------------------------------------------------------
    import React from 'react';
    
    import myStyles from './myStyle.module.css';                                       // -! imported stylesheet ID and Class are only available in this module (other element styles are still available in the whole HTML document)   
    
    class ChildComponent extends React.Component{
        render(){
            return (
                <div>
                    <p className='{myStyles.myClass}> Child Element-1 </p>             // class from './myStyle.css'
                    <p id={myStyles.myId} > Child Element-2 </p>                       // id from './myStyle.css'
                </div>
            )
        }
    }
    
    export default ChildComponent;

Inline Styling in JSX

DEMO 
    import React from 'react';
    import ReactDOM from 'react-dom';
    
    class TestComponent extends React.Component{
        render(){
            let style_1 = {color:'red', padding:'10px'}
            return(
                <div>
                    <p style={{color:'red', padding:'10px'}}>some element</p>          // inline styles are passed in a JavaScript object   
                    <p style={style_1}>some other element</p>                          // style object referenced   
                </div>
            )
        }
    }
    
    ReactDOM.render(<TestComponent />, document.getElementById('testElement'));

Referencing CSS IDs and Classes from JSX

DEMO 
    import React from 'react';
    import ReactDOM from 'react-dom';
    
    class TestComponent extends React.Component{                                     
        render(){   
            return(
                <div>
                    <p className="reactClass">Element 1</p>                            // CSS calsses are referenced as className in JSX
                    <p className="reactOtherClass">Element 2</p>
                    <p id="reactId">Element 3</p>                                      // CSS ids are referenced as id in JSX (just like in normal HTML)
                </div>
            )
        }
    }
    
    ReactDOM.render(<TestComponent />, document.getElementById('testElement'));
