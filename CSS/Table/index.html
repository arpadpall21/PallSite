Table Styling [Ordered: Alphabetically]
Property Value Description
border-collapse separate (Default: separate borders)
collapse (collapse borders into a single one)
 - defines if the border of the <table> element should collapse with its children
- used by <table> / <inline-table> elements 1 1 4 5 1.4
border-spacing length (Default: 2px, space between cells)
lengthHor lengthVert (space between cells separately)
 - defines the space between table cells
- used by <table> / <inline-table> elements 1 1 4 8 1
caption-side top (Default: above the table)
bottom (below the table)
 - defines the caption position
- used by <table> / <inline-table> / <caption> elements 1 1 4 8 1
empty-cells show (Default: show borders)
hide (hide borders)
 - defines whether show or hide empty table cell borders
- used by <table> / <inline-table> / <td> / <th> / <tr> elements 1 1 4 8 1.2
table-layout auto (Default: renders the whole table at once)
fixed (renders the table dinamically)
 - defines the rendering method (show part what is already downloaded or not)
- used by <table> / <inline-table> elements 14 1 7 5 1

Cell Spaning

Test Table (caption)
Ab B
1 2 3
4 5 6
7 8
CODE : 
-- CODE (HTML) --
    <table class="first">
    <caption> Test Table (caption)</caption>
        <tr>
            <th> Ab </th>
            <th colspan="2"> B </th>                // span
        </tr>
        <tr>
            <td> 1 </td>
            <td> 2 </td>
            <td> 3 </td>
        </tr>
        <tr>
            <td rowspan="2"> 4 </td>                // span
            <td> 5 </td>
            <td> 6 </td>
        </tr>
        <tr>
            <td> 7 </td>
            <td> 8 </td>
        </tr>
    </table>
    
-- CODE (CSS) --    
    .first {
        border: 3px solid black;
        background-color: yellow;
        border-collapse: collapse;
        text-align: center;
        width: 80%;
        margin: auto;
        padding: 5px;
    }
    .first th{
        background-color: red;
        border: 1px solid black;
    }
    .first td{
        border: 1px solid black;
    }

Placing "caption" and removing empty cells

Caption on the bottom
-A- -C-
-2- -3-
-1- -3-
-1- -2-
CODE : 
-- CODE (HTML) --
    <table class="style_style">
    <caption> Caption on the bottom </caption>
        <tr>
            <th> -A- </th>
            <th>     </th>
            <th> -C- </th>
        </tr>
        <tr>
            <td>     </td>
            <td> -2- </td>
            <td> -3- </td>
        </tr>>
        <tr>>
            <td> -1- </td>
            <td>     </td>
            <td> -3- </td>
        </tr>
        <tr>
            <td> -1- </td>
            <td> -2- </td>
            <td>     </td>
        </tr>
    </table>
    
-- CODE (CSS) --
    .style_style{
        margin: auto;
        border: solid 1px black;
        background-color: crimson;
        caption-side: bottom;               // <- caption position
        border-spacing: 15px;
        empty-cells: hide;                  // <- remove empty cells
    }
    .style_style th,.style_style td{
        border: solid 1px black;
        margin: auto;
        padding: 10px 40px;
        background-color: forestgreen;
    }
    .style_style caption{
        font-size: 1.5em;
        text-decoration: underline;
    }
