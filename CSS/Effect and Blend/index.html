the filter property 

SYNTAX :    filter: filterVal filterVal ...                     //  defines the visual effect to the element 
                    none               // (default) no effect
                    blur(nrpx)         // 0- [default 0]
                    brightness(nr%)    // 0-100 [default 100]
                    contrast(nr%)      // 0- [default 100]
                    drop-shadow(hShadowpx, vShadowpx, blurpx, spreadpx, color)   // drop shadow effect
                    greyscale(nr%)     // 0-100 [default 0]
                    hue-rotate(nrdeg)  // color on the hue wheel (0-360 [default 0])
                    invert(nr%)        // sample invert (0-100 [default 0])
                    opacity(nr%)       // 0-100 [default 100]
                    saturate(nr%)      // 0- [default 100]
                    sepia(nr%)         // sepia conversion (0-100 [default 0])
                    url(URL#filter)    // applies an SVG filter (URL points to an SVG <filter> element)
    
    
    #demo {
        filter: opacity(50%) saturate(50%);                     // image half opaque and half desaturated
    }

the background-blend-mode / mix-blend-mode / isolation properties 

SYNTAX :    background-blend-mode: normal // (default)          //  blending mode between multiple background layers 
                                   normal
                                   multiply
                                   screen
                                   overlay
                                   darken
                                   lighten
                                   color-dodge
                                   saturation
                                   color
                                   luminosity
                                   
            mix-blend-mode: normal     // (default)             //  specifies how the element's content blends with the direct background (any backgrounds directly behind the content)
                            multiply                    
                            screen                      
                            overlay                     
                            darken                      
                            lighten                     
                            color-dodge                 
                            color-burn                  
                            difference                  
                            exclusion                   
                            hue                         
                            saturation                  
                            color                       
                            luminosity
                            
            isolation: auto            // (default) blends through   //  this element isolates children elements from blending with backdrop elements relative to this element (isolator middle element) (children elements still blend with this element)  
                       isolate         // isolate children form backdrop elements 
    
-- HTML -- 
    <div class="bgBlend">
        <img src="Images/Loch Ness.jpg" alt="Loch">
    </div>
    
-- CSS --
// background-blend-mode ------------------------------------------------- 
  // -! backgrounds cannot be isolated with the "isolate" property, because we cannot insert ani isolater element between them
    .myElement {
        background-color: red;
        background-image: url(Images/lightning.jpg);
        background-blend-mode: multiply;                        // blends the two background together   
    }
    
// mix-blend-mode --------------------------------------------------------  
    .myElement {
        background-color: red;   
    }
    .myElement img {                                            // the "img" element is the children (content) of the "myElement" 
        mix-blend-mode: multiply;                               // blends with the direct background    // -! ANY background what is directly behind the element 
    }
    
// isolation ------------------------------------------------------------- 
  // CASE-1 ------------- 
    .bgBlend {
        background-color: red;                                  // -! children elements will sill blend with this (isolator) element 
        isolation: isolate;                                     // isolate the children elements form backdrop elements 
    }
    .bgBlend img {
        mix-blend-mode: multiply;                               // blends with the isolator element's background only!
    }
    
  // CASE-2 ------------- 
    .bgBlend {                                                  // this element has no background set 
        isolation: isolate;                                     // isolate the children elements form backdrop elements 
    }
    .bgBlend img {
        mix-blend-mode: multiply;                               // no blend because the isolator element has no background 
    }

isolation property TEST

- the blending mode is set on the grand-child element (element with the image)

- the child element (element with blue border) is where the isolation:isolate; is set (this layer isolates the grand-child element)

square-1: the middle square (blue border) has no background set so the grand-child element has nothing to blend with

square-2: the middle square (blue border) has a green background set so the grand-child element blend with it but not with the parent


- here is how this works the element which has the isolation:isolate; set will isolate all children element form its parent element form any blending but the element intself will still blend with its children elements

CODE : 
-- CODE (HTML) -- 
    <div class="isolate">
        <div class="isolateCh_1">
            <div class="isolateGrandCh_1"></div>
        </div>
        <div class="isolateCh_2">
            <div class="isolateGrandCh_2"></div>
        </div>
    </div>
        
-- CODE (CSS) -- 
    .isolate {                                          // parent element 
        width: 400px;
        height: 200px;
        border: solid 2px black;
        display: flex;
        background-color: red;
    }
    .isolateCh_1 {                                      // isolator element 1
        display: flex;
        margin: auto;
        width: 150px;
        height: 150px;
        border: solid 3px blue;
        isolation: isolate;                             
    }
    .isolateGrandCh_1 {
        margin: auto;
        width: 100px;
        height: 100px;
        border: solid 2px black;
        background-image: url(Images/rock.jpg);
        background-size: cover;
        mix-blend-mode: screen;
    }
    .isolateCh_2 {                                      // isolator element 2
        display: flex;
        margin: auto;
        width: 150px;
        height: 150px;
        border: solid 3px blue;
        background-color: green;
        isolation: isolate;
    }
    .isolateGrandCh_2 {
        margin: auto;
        width: 100px;
