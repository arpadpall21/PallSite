Description and Demonstration :

- Media Queries are an efficient technique for delivering tailored styles for different media types and screen sizes

- There are 2 ways to implement media queries

1) we can implement media queries on the stylesheet with the @media at rule

2) media query conditions can be passed as media attribute in the stylesheet link element (so the stylesheet is applied only if the media query conditions passes) 

// in head link 'media' attribute -----------------------------------------------------
    <head>
        <link rel="stylesheet" media="only print" href="print.css">                    // sheet only implemented on print view     
        <link rel="stylesheet" media="(max-width: 800px)" href="max800.css">           // sheet only implemented on devices less than 800 pixels
    </head>
    
// @media (in CSS stylesheet) ---------------------------------------------------------
    @media screen and (max-width:500px) {                                              // applies styles to screens that are less than 500px width    
        // CSS code....
    }

DEMO 
    @media print { ... }                                                               // runs the block if the view is in print mode 
    @media screen, print {}                                                            // multiple query conditions (only one of the must be true)
    
    @media screen and (max-width: 12450px) { ... }  
    @media screen and (min-width: 30em) and (orientation: landscape) { ... }  
    @media (min-height: 680px), screen and (orientation: portrait) { ... }             // multiple query conditions (only one of the must be true)

    (stylesheet exclusive for pirnting)
    <link rel="stylesheet" media="only print" href="styles-media.css">

    }

simple example

(width=1000px) Background Color is green between 500px and 800px
(width=1000px) Background Color is green below 500px and above 800px
CODE : 
-- HTML --
    <div class="media-one"> (width=1000px) Background Color is green between 500px and 800px </div>
    <div class="media-two"> (width=1000px) Background Color is green below 500px and above 800px </div>
                
-- CSS --
    [class|=media]{
        background-color: orange;
        max-width: 1000px;
        height: 50px;
        margin: auto;
        text-align: center;
        border: 3px black solid;
    }
    @media screen and (max-width:800px) and (min-width:500px){
        .media-one{
            background-color: green; 
        }
    }
    @media screen and (max-width:500px), screen and (min-width:800px){
        .media-two{
            background-color: green;
        }
    }
