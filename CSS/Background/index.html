Remember This :

- when set multiple background images in a single property, the leftmost images are the closest to the viewer

- you can use CSS gradient functions anywhere you can use images (background, border, ect..)

Description and Demonstration :

- in HTML we can set a background by using various properties to nearly any elements, even for the html tag

- it is possible set multiple background images, when we set in a single property value, the leftmost images are the closest to the viewer

Using Gradients as Backgrounds

The linear-gradient() / repeating-linear-gradient() / radial-gradient() / repeating-radial-gradient() / conic-gradient() properties

- gradient functions generate special kind of images what can be used as property values anywhere where images can be uses (background, borders, etc...)

DEMO 
// HTML -------------------------------------------------------------------------------
    <div class="demo"></div>
    
// CSS --------------------------------------------------------------------------------
    .demo {
        width: 20%;
        height: 150px;
  // linear gradient -------------------------
        background: linear-gradient(to top, red , white)                               // linear gradient bottom-top
        background: repeating-linear-gradient(90deg, red 5%, white 10%);               // color stops are specified but repeated to fill the background
  // radial gradient -------------------------
        background: radial-gradient(circle farthest-side at 100% 100%, blue, red);     // circle shaped radial gradient goes from bottom right to the "farthest-sie" of the element
        background: repeating-radial-gradient(blue 2%, red 4%);                        // ellipse shaped radial gradient placed in the center (default values), color stops are specified but repeated to fill the background of the element  
  // conic gradient --------------------------
        
    }
