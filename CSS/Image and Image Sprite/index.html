Image Sprites

- an image sprite is a collection of images put into a single image

- an image sprite is basically showing one part of an image as background in an element 

-- HTML --    
    <span id="demo"></span>
    
-- CSS --    
    #demo {
        display: inline-block;
        width: 200px;                                           // element's dimension set (background part is displayed at this size)
        height: 200px;
        background: url(Images/lightning.jpg) 100px 100px;      // background image and position set (the two coordonates define the background image top left hanging corner) 
    }

the filter property 

SYNTAX :    filter: filterVal filterVal ...                     //  defines the visual effect to the element 
                    none               // (default) no effect
                    blur(nrpx)         // 0- [default 0]
                    brightness(nr%)    // 0-100 [default 100]
                    contrast(nr%)      // 0- [default 100]
                    drop-shadow(hShadowpx, vShadowpx, blurpx, spreadpx, color)   // drop shadow effect
                    greyscale(nr%)     // 0-100 [default 0]
                    hue-rotate(nrdeg)  // color on the hue wheel (0-360 [default 0])
                    invert(nr%)        // sample invert (0-100 [default 0])
                    opacity(nr%)       // 0-100 [default 100]
                    saturate(nr%)      // 0- [default 100]
                    sepia(nr%)         // sepia conversion (0-100 [default 0])
                    url(URL#filter)    // applies an SVG filter (url points to an SVG filter element)
DEMO 
    
    #demo {
        filter: opacity(50%) saturate(50%);                     // image half opaque and half desaturated
    }

the object-fit / object-position properties 

SYNTAX :    object-fit: fill           // (default) scaled to fill the content-box (stretched/squished to fit)   //  specifies how an <img> or <video> "own content" is sized on its own container  
                      : contain        // maintains its aspect ratio and scaled to fill the content-box "letterboxed" 
                      : cover          // maintains its aspect ratio and scaled to fill the whole area of the content-box  
                      : none           // content not sized (keep its original size) 
                      : scale-down     // none or contain whichever gives the smallest value 
                      
            object-position: top|right|bottom|left|center       //  positions an <img> or <video> "own content box" inside its own container (used with the object-fill property)  
                           : XcssLength YcssLength
DEMO 
    img {
        width: 800px;                                           // we set the container size bigger than the original image 
        height: 500px;
        object-fit: none;                                       // the image itself (content-box) is not scaled with its container 
        object-position: top right;                             // the image itself (content-box) is positioned 
    }
Example :

object-fit property TEST

- the red border is the original border of the image (scaled up)

- object-fit:fill (default) sized to fill the element's content

- object-fit:contain image (own content) mantains its aspect ratio, scaled to fitt the elements content box "letterboxed"

- object-fit:cover image (own content) maintains its aspect ratio, scaled to fill to whole area of the content box

- object-fit:none image (own content) is not sized

- object-fit:scale-down choses between the none and contain values whichever gives the smallest result

- on the first image it choses the value none because the image container is bigger than the image itself

- on the second image it choses the value contain because the image container is smaller the the image itself

CODE : 
-- HTML -- 
    <img class="objFit posFill" src="Images/orange_flowers.jpg" alt="orange flowers">
    
    <img class="objFit posContain" src="Images/orange_flowers.jpg" alt="orange flowers">
    
    <img class="objFit posCover" src="Images/orange_flowers.jpg" alt="orange flowers">
    
    <img class="objFit posNone" src="Images/orange_flowers.jpg" alt="orange flowers">
    
    <img class="objFit posScaleDown" src="Images/orange_flowers.jpg" alt="orange flowers">
    <img class="posScaleDown_" src="Images/orange_flowers.jpg" alt="orange flowers">
    
-- CSS -- 
    .objFit {
        border: solid 1px red;
        width: 200px;
        height: 200px;
    }
    .posFill { object-fit: fill }
    .posContain { object-fit: contain }
    .posCover {object-fit:cover }
    .posNone {object-fit:none }
    
// scale-down --------------------------------------------------- 
    .posScaleDown {
        object-fit: scale-down 
    }
    .posScaleDown_ {
        border: solid 1px red;
        width: 80px;
        height: 80px;
        object-fit: scale-down;
    }
