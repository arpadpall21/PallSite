Remember This :

Description and Demonstration :

Margins

- margin is the space around the element outside of any defined borders

- the margin cannot have background and accept negative values (elements can overlap therefore) 

SYNTAX :    margin-top: cssLength                                  // sets the top margin of the element
                        auto           // browser calculates (horizontally centers) 
            margin-right: cssLength                                // sets the right margin of the element
                          auto         // browser calculates (horizontally centers) 
            margin-bottom: cssLength                               // sets the bottom margin of the element
                           auto        // browser calculates (horizontally centers) 
            margin-left: cssLength                                 // sets the left margin of the element
                         auto          // browser calculates (horizontally centers) 
            
            margin: top right bottom left  // 4 values             // margin shorthand property 
                    top leftRight bottom   // 3 values  
                    topBottom leftRight    // 2 values
                    all                    // 1 value
    
       
    margin-top: 100px;                                             // margin top is 100px
    margin-right: 2em;                                             // margin right is twice as bit as the element's letter size
    margin-bottom: 5%;                                             // bottom margin is 5% of the parent element's width
    margin-left: inherit;                                          // uses the parent element's left margin paremeters
    
    margin: 5px;                                                   // all margins are set to 5px
    margin: 2% 10%;                                                // top and bottom margins are 2% of the parent element's width and left and right margins are 10% of the parent element's width
    margin: 10px 4em 5px;                                          // top margin = 10px / left and right margins = 4em / bottom margin = 5%
    margin: 10px 8px 5px 2px;                                      // top margin = 10px / right margin = 8px / bottom margin = 5% / left margin = 2%

margin collapse TEST

margin right 15pxmargin left 30px = margin between the elements is 30px because the highes margin is honored only!

margin right -15pxmargin left 20px = margin between the elements is 5px because the negative value is subtracted form the positive!

I have a -10px bottom margin (margin kept because of the highest value)

I have a -9px top margin

CODE : 
-- HTML --
// only the 30px margin kept
    <p class="marginCollapse"> <span style="margin-right:15px;">margin right 15px</span><span style="margin-left:30px;">margin left 30px</span> = margin between the elements is 30px because the highes margin is honored only!</p>
    
// margin here is 5px between element, in this situation (positive and negative values) an arithmetic operation is accomplished
    <p class="marginCollapse"> <span style="margin-right:-15px;">margin right -15px</span><span style="margin-left:20px;">margin left 20px</span> = margin between the elements is 5px because the negative value is subtracted form the positive!</p>   
    
// only the -10px margin is kept
    <p style="background-color:white; border-bottom:1px black solid; margin-bottom:-10px;"> I have a -10px bottom margin (margin kept because of the highest value)</p>
    <p style="background-color:white; border-bottom:1px black solid; margin-bottom:-9px;"> I have a -9px top margin </p>
    
-- CSS ---
    .marginCollapse span {
        border: solid 1px black;
        background-color: white;
    }

Margin Collapse

- if two elements have adjacent margins (even if parent / children) these margins will collapse (except if the element is floating or its position is absolute)

- the collapse mechanism works very strangely (some consider it as a Bug), the browser keeps the margin that has a value farther from 0 or subracts the negative value from the positive (if values are negative / positive)

-- HTML --
    <p class="frist"> first element </p>
    <p class="second"> second element </p>
    
-- CSS --
// margin between elements is 60px (number farther from 0 is honored)
    .first {margin-bottom: 50px;}
    .second {margin-top: 60px;}
    
// margin between elements is -10px (number farther from 0 is honored)
    .first {margin-bottom: -10px;}
    .second {margin-top: -7px;}
    
// margin between elements is -5px (negative is subtracted from positive margin)
    .first {margin-bottom: 15px;}
    .second {margin-top: -20px;}

Paddings

- the padding is the space around the element's content inside of any defined border

- the padding is affected by the background and does not accept any negative values! 

SYNTAX :    padding-top: cssLength                                 // sets the top padding of the element
            padding-right: cssLength                               // sets the right padding of the element
            padding-bottom: cssLength                              // sets the bottom padding of the element
            padding-left: cssLength                                // sets the left padding of the element
            
            padding: top right bottom left  // 4 values            // padding shorthand property 
                     top leftRight bottom   // 3 values  
                     topBottom leftRight    // 2 values
                     all                    // 1 value  

    
    padding-top: 100px;                                            // padding top is 100px
    padding-right: 2em;                                            // padding right is twice as bit as the element's letter size
    padding-bottom: 5%;                                            // bottom padding is 5% of the parent element's width
    padding-left: inherit;                                         // uses the parent element's left padding paremeters
    
    padding: 5px;                                                  // all paddings are set to 5px
    padding: 2% 10%;                                               // top and bottom paddings are 2% of the parent element's width and left and right paddings are 10% of the parent element's width
    padding: 10px 4em 5px;                                         // top padding = 10px / left and right paddings = 4em / bottom padding = 5%
    padding: 10px 8px 5px 2px;                                     // top padding = 10px / right padding = 8px / bottom padding = 5% / left padding = 2%
