CSS Counters

- CSS Counters are variables maintained by CSS 

SYNTAX :    counter-reset: none         // (default) does nothing               // creates new or sets an existing CSS counter
                         : id nr        // sets the counter identifier and set its starting value (default 0)

            counter-increment: none     // (default) does nothing               // defines the increment/decrement value for a css counter
                             : id nr    // defines the increment/decrement value for the targeted css counter (default increment 1))

            content: counter(counterId)                                         // returns the cssCounter value for the current element (this is a "content" property value)
                   : counters(counterId, "nestSeparator")                       // returns the cssCounter value for the current element (handles nested elements) (this is a "content" property value)  
                        - counterId : css counter identifier
                        - "nestSeparator" : defines a separator character between nested counters

- counter() function simply returns the number of targeted elements

-- CSS --
    html {
        counter-reset: simpleCount 0;               // counter variable can be declared (and initialized) nearly anywhere in the case of the "counter()" function 
    }
    .demo li::before {
        counter-increment: simpleCount 1;           // incremet the counter value by "1" at every element 
        content: " " counter(simpleCount);          // the "content" property insert the returned value by the "counter()" function
    }

Insert content with CSS content 

SYNTAX :    content: normal             // (default) same as "none"             // used with the ::before and ::after pseudo elements to insert a generated content
                   : none               // no content
                   : counter(counter_Id)// css counter result
                   : "text"             // text
                   : attr("attrId")     // targeted attribute's value
                   : open-quote         // add opening quote
                   : close-quote        // add closing quote
                   : no-open-quote      // removes opening quote
                   : no-close-quote     // removes closing quote
                   : url("URL")         // media file
                   : value value ..     // multiple values are allowed
    
    <p> myContent </p> 
                                  
-- CSS --
    p::before {
        content: " -->"                 // adds the " -->" text before the targeted paragraph
        color: red;                     // newly added content styled here
    }
    
-- display in HTML --
    --> myContent
