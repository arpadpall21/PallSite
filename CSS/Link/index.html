simple example

- This link goes to the W3Schools site (also styled)

W3SCHOOLS.COM
CODE : 
    -- CODE (HTML) --
    <a id="link" href="http://www.w3schools.com" title="W3Schools" target="_blanc"> W3SCHOOLS.COM </a>
    
    -- CODE (CSS) --
    #link:link{                                                 #link:hover{
        color: purple;                                              color: black;
        background-color: black;                                    background-color: yellow;
        text-decoration: underline;                                 text-decoration: overline;
    }                                                           }
    #link:visited{                                              #link:active{
        color: black;                                               color: white;
        background-color: lightgreen;                               background-color: red;
        text-decoration: underline;                                 text-decoration: underline;
    }                                                           }

image link

- This image is a link it leads to Youtube (changes when hover, it needs a fancy code for achive this)

CODE : 
    -- CODE (HTML) --
    <p align="center"><a id="link_image" href="https://www.youtube.com" target="_blanc" title="Youtube"></a></p>
    
    -- CODE (CSS) --
    #link_image{
        display: block;
        width: 50px;
        height: 25px;
        background-image: url(../../../../Pictures/Youtube.png);
        background-size: 100% 100%;
        background-repeat: no-repeat;
        transition: .2s;
    }
    #link_image:hover{
        background-image: url(../../../../Pictures/Youtube-hover.png);
    }

style link with CSS

YAHOO

CODE : 
    -- CODE (HTML) --
    <a id="button" href="https://fr.yahoo.com" title="Yahoo-English" target="_blanc">YAHOO</a>
    
    -- CODE (CSS) --
    #button:link, #button:visited{
        padding:15px 25px;
        background-color:green;
        color:white;
        text-decoration:none;
        border-radius: 8px;
    }
    #button:hover, #button:active{
        background-color:red;
    }
