Border Image

border-image / border-image-outset / border-image-repeat / border-image-slice / border-image-source / border-image-width

- setting image as borders we fist must set a transparent border, (the image border does not affect the layout!)

- the image border system takes an image and slices it into nine section (like a tic-tac-toe board) than handles these sliced parts as border images 

SYNTAX :    border-image-source: none                          // default                      //  specifies the image for the border image
                               : url("URL")
                               
            border-image-slice: nr(px)|nr%                     // sets the slice coordinates counted from the image edges (4 sides equally)    //  specifies how to slice the border-image-source   
                              : nrH(px)|nrH% nr(px)V|nrV%      // sets the slice coordinates counted from the image edges (horizontal and vertical coordinates are controled separately)
                              : sliceParameters fill           // displays the middle part of the border-image-source
            
            border-image-width: top right bottom left          // 4 values                     //  specifies the border image width (image border extends towards the element center without affecting the padding and content)  
                              : top leftRight bottom           // 3 values  
                              : topBottom leftRight            // 2 values
                              : all                            // 1 value
                              : nr                             // relative to the "border-width" property (default 1)     
                              : nr%                            // relative to the element's dimensions 
                              : nrpx
                              : auto                           // uses the width of "border-image-slice" value if available, otherwise it uses the "border-width" value 
                              
            border-image-repeat: stretch                       // (default) the side slice is stretched to fill the side border     //  specifies how to handle the side border image slice
                               : repeat                        // the side slice is repeated (without stretching) to fill the side border   
                               : round                         // the side slice is repeated and stretched to perfectly fill the side border  
                               : space                         // the side slice repeated (without stretchng) the remaining spaced are distribute evenly (does not work as expected accross the browsers [TESTED 2019-01-08])   
                        
            border-image-outset: nr                            // relative to the "border-width" property (default 0)      //  extends the border-image beyond the border box (without affecting the layout) 
                               : cssLength 
                               
            border-image: source slice width outset repeat                                     //  border-image shorthand property (works only with "source slice repeat" [TESTED 2019-01-09]) 
    
    <p class="demo"> TEST </p>
    
-- CSS --
    .demo {
        border: 10px solid transparent;                        < -- transparent border must be set first!
        border-image-source: url("myImage.png");
        border-image-slice: 100 75;                            // slice horizontal lines are 100 pixels from top and bottom and vertical lines are 75 pixels form left and right
        border-image-width: 2;                                 // border width doubled towards inside without affecting the padding (relative to the original "border-width" (2x10 = 20px)
        border-image-repeat: round;                            // side image borders are repeted and "rounded" (stretched) in order to fill the side border image pattern perfectly
        border-image-outset: 3;                                // border image extended out 30px (border with 10px = 10x3) from the original border (without affeting the layout)
    // shorthand property ----------------------------------
        border-image: url("myImage.png") 100 round;            // "source slice repeat" properties set (tried to add "width outset" but it is not working [TESTED 2019-01-09])
    }

Rounded Corners

border-radius

- CSS3 gives the ability to round the elements' corners

- "border-radius" properties actually modify the box shape of the element 

SYNTAX :    border-radius: topLeft topRight bottomRight bottomLeft  // 4 values                //  shorthand property for the border radius properties 
                         : topLeft topRightBottomLeft bottomRight   // 3 values  
                         : topLeftBottomRight topRightBottomLeft    // 2 values
                         : all                                      // 1 value
                         
            border-top-left-radius:     | cssLength                                            //  rounds the element's targeted corner
            border-top-right-radius:    | nr%                  // relative to the element's dimensions 
            border-bottom-left-radius:  |
            border-bottom-right-radius: |
    
    <div class="demo"></div>
    
-- CSS --   
    .demo {
        width: 200px;
        height: 200px;
        border: solid 5px black;
        border-top-left-radius: 25%;
        border-top-right-radius: 20%;
        border-bottom-right-radius: 15%;
        border-bottom-left-radius: 10%;
    // shorthand property does the same as the 4 "border-radius" properties do above -----------------
        border-radius: 25% 20% 15% 10%;
    }

Outline

outline / outline-color / outline-style / outline-width / outline-offset

- the outline is a "sort of border" around the element, which does not affect the layout (outest line)

- unlike borders we cannot control separate sides of the outline 

SYNTAX :    outline-color: invert                              // (default) inverts the background color [IE only] other browsers sets the default color to black       // sets the color of the outline  
                         : cssColor        
        
            outline-style: none                                // (default)                    // sets the style of the outline
                         : hidden
                         : dotted
                         : dashed
                         : solid
                         : double
                         : groove
                         : ridge
                         : inset
                         : outset
                         
            outline-width: medium                              // default                      // sets the outline width   
                         : thin 
                         : thick
                         : cssLength
                         
            outline-offset: cssLength                          // default 0                    // adds space between the element and the outline
            
            outline: width style color                                                         // outline shorthand property
    
    <div class="demo"></div>
    
-- CSS --
    .demo {
        width: 200px;
        height: 200px;
        outline-color: red;
        outline-style: solid;
        outline-width: 10px;
        outline-offset: 5px;
    // shorthand property does the same ----------------
        outline: 10px solid red;
        outline-offset: 5px;                                   // shorthand property does not support the "outline-offset" therefore it is set separately
    }
