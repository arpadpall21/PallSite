Description and Demonstration :

- in CSS there are 2 types of font family names generic family and font family

- every font falls into 1 of 5 "generic family" serfi / sans-serif / monospace / cursive / fantasy

- Web Safe Fonts are font families what are preinstaled on the user's system> Web Safe Fonts

- I is recommended to declare the "generic font family" at the end of every "font-family" declaration because the browser uses it as fallback 

SYNTAX :    font-family: family-name ... generic-family                // specifies the font family 
    
            font-size: medium           // (default 16px               // sets the size of the font

The @font-face rule

- the @font-face roule makes possible to use external fonts stored in files on the web page, there are several font formats check compatibility W3Schools Web Fonts 

SYNTAX :    @font-face {
                font-family: customName;                               // custom name to the added font 
                src: url(URL_to_the_font_file);                        // path to the font file
                font-stretch: value;
                font-style: value;
                font-weight: value;
                unicode-range: value;
            }    
            
            unicode-range: U+unicodePoint-unicodePoint, ...   // default U+0-10FFFF    // set a character or a specific range of characters to be used from the font defined by the font-face roule, (used for performance imporuvement, browser only download the specified characters)   
    
    @font-face {
        font-family: myFont;                            // set a name (identifier) 
        src: url(Font-Families/GloriaHallelujah.ttf);   // reference the font file
        font-weight: 300;
        unicode-range: U+0-00A0;                        // only characters from unicode "0" to unicode "00A0" are downloaded 
    }

    .demoElement {
        font-family: myFont, serif;                     // using the downloaded font 
    }
