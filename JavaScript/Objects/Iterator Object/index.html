6 Iterator Object [Ordered: Alphabetically]
Method Description
iteratorObj - created by a generator function or by various methods (ex: mapObj.entries()) from ES6
- holds its items in strict iterable order
- its items can be assecced by the next() method or by using the for... of loop support depend what created it
iteratorObj.next() - returns the next item of the iterable object in an object like {value:val, done:bol}
- if the done property is true the iterator object is finished support depend what created it

Description and Demonstration :

- the Iterator object is an Array-like object which holds its items in strict order, we can access its items through the next() method or by using the for... of loop

- the Iterator object can be created by various tools, like generator functions or by various methods (ex: mapObj.entries())

DEMO 
    var nrMap = new Map([[1, "egy"], [2, "ketto"], [3, "harom"]]);
    var iTem = nrMap.keys();                                                           // iterator object 
    
    var gen = function*(){
        yield [1, "egy"]; 
        yield [2, "ketto"];
        yield [3, "harom"];
    }
    var iGen = gen();                                                                  // iterator object 
    
// access items -----------------------------------------------------------------------
    iTem.next();                                                                       // -> {value:1, done:false}    
    iTem.next();                                                                       // -> {value:2, done:false}
    iTem.next();                                                                       // -> {value:3, done:false}                
    iTem.next();                                                                       // -> {value:undefined, done:true}   // iterator object finished  
    iTem.next();                                                                       // -> {value:undefined, done:true}   // iterator object won't restart   
    
    for(let i of iGen) {
        i;                                                                             // -> [1, "egy"] | [2, "ketto"] | [3, "harom"]
    }
