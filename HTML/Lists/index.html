Unordered list <ul>

- the <ul> defines an unordered (bulleted) list along with the <li> (item) tag 

SYNTAX :    <ul>
                <li> text </li>
                <li> text </li>
                <li> text </li>
            </ul>
            
    // <ul> element attributes - - - - -
        type="disc/square/circle           // specifies the marker type(no HTML5 support)
    
    <ul>
        <li> Samsung </li>
        <li> Sony </li>
        <li> Apple </li>
    </ul>

Ordered list <ol>

- the <ul> defines an ordered (numerical or alphabetical) list along with the <li> (item) tag 

SYNTAX :    <ol>         
                <li> text </li>                           
                <li> text </li>                         
                <li> text </li>
            </ol>
            
    // <ol> element attributes - - - - -            
            reversed (boolean)             //  reverses the order of the markers  (No EDGE support)
            start="nr"                     // defines where the marker count start from     
            type="1/A/a/I/i"               // specifies the marker style 
    
    <ol type="1" reversed >                              // count reversed (does not revers list items) 
        <li> Samsung </li>
        <li> Sony </li>
        <li> Apple </li>
    </ol>

Drop-Down List <select> / <option> / <optgroup>

SYNTAX :    <select>                                     // creates a drop-down list
                <optgroup lable="myLabel">               // defines an group for the options
                    <option> text <option>               // option list
                    <option> text <option>
                    <option> text <option>
                </optgroup>    
            </select>
            
    // <select> element attributes - - - - -      
        autofocus                          //  gets focus automatically when page loads 
        disabled                           // disables the drop-down list
        form="form_id"                     //  defines one or more forms the list belongs to 
        multiple                           // multiple options can be selected (hold ctrl+click)
        name                               // defines a name for the drop-down list
        required                           //  an item with a valid value must be selected before submitting  (No Safari support)  
        size                               // defines how many items (options) should be visible 
          
    // <option> element attributes - - - - -
        disabled                           // (boolean) disables the list item 
        label="text"                       // label to be printed instead of the typed value (no Mozilla support) 
        selected                           // (boolean) preselects the item                
        value="text"                       // value sent to the server   
        
    // <optgroup> element attributes - - - - -  
        disabled                           // (boolean) disables the option group 
        label="text"                       // defines an optiongroup label 
    
    <select required form="myForm" autofocus> 
        <optgroup label="germanCars" >                               // option group defined
            <option value="audi"> Audi </option>
            <option value="bmw" selected> BMW </option>              // preselected item
            <option value="mercedess" disabled> Mecedess </option>   // disabled item
        </optgroup>
        <optgroup label="other Cars" disabled>                       // option group disabled                      
            <option value="volga" > Volga </option>
        </optgroup>    
    </select>

Datalist <datalist> 

SYNTAX :    <datalist id="target_input" >                // creates a pre-defined list for an "input" element (sort of autocomplete list) (no Safari support)
                <option value="text">
                <option value="text">
                <option value="text">
            </datalist>
    
    <input type="text" list="numbers">                   // the datalist submits through the "list" attribute  
    <input text="number" list="numbers">                 <-- Firefox datalist does not support numbers!
    
    <datalist id="numbers">                              // same list used for to different inputs
        <option value="111">
        <option value="112">                             // closing tag not required
        <option value="112">
    </datalist>
