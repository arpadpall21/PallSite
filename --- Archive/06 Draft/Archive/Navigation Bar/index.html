Remember This :

- Declare elements as block displayed that way they can have their paddings and margins!

- you can set the width of ul item this creates a container for its children therefore no need to set childrens widht!

- in inline displayed based horisontal list the li MUST be declared display: inline-block

- in floating based horizontal list the ul element MUST be declared overflow: hidden; this prevents li elements going outside from the list

- the float based horizontal menu is better

Some Description :

The base of all list displayed menus

- 1) we use an HTML list as the base element of a navigation bar

- 2) remove the bullets form the list list-style-type: none;

- 3) set margin and padding to 0, margin:0; padding:0;

Vertical Navigation Bar

- There are 2 ways to create list based horizontal menu, by using Inline or Float list elements

- In this case I have found that the float method is better in several ways

Inline method based horizontal menu

  • Home
  • First Item
  • Second Item
  • Third Item
  • Fourth Item
    -- CODE (HTML) --
    <ul class="inlineblock">
        <li><a id="active" href="Home">Home</a></li>
        <li><a href="fist item">First Item</a></li>
        <li><a href="second item">Second Item</a></li>
        <li><a href="third item">Third Item</a></li>
        <li><a id="last" href="fourth item">Fourth Item</a></li>
    </ul>
    
    -- CODE (CSS) --
    ul.inlineblock{                                     ul.inlineblock li a{
        list-style-type: none;                              text-decoration: none;
        padding: 0;                                         font-weight: 700;
        margin: 0;                                          color: black;
        background-color: gray;                             display: block;
    }                                                       padding: 13px;
    ul.inlineblock li{                                      padding-right: 11px;
        display: inline-block;                          }
        margin-right: -10px;                            ul.inlineblock li a#active{
    }                                                       background-color: green;
                                                        }
                                                        ul.inlineblock li:hover:not(#active){
                                                            background-color: white;
                                                        }

Float method based horizontal menu

  • Home
  • First Item
  • Second Item
  • Third Item
  • Fourth Item
    -- CODE (HTML) --
    <ul class="inlinefloat">
        <li><a id="active" href="Home">Home</a></li>
        <li><a href="fist item">First Item</a></li>
        <li><a href="second item">Second Item</a></li>
        <li><a href="third item">Third Item</a></li>
        <li id="last"><a id="last" href="fourth item">Fourth Item</a></li>
    </ul>
    
    -- CODE (CSS) --
    ul.inlinefloat{                                             ul.inlinefloat li a#active{
        list-style-type: none;                                      background-color: green;
        padding: 0;                                             }
        margin: 0;                                              ul.inlinefloat li#last a#last{
        overflow: hidden;                                           background-color: rgb(25,25,25);
        background-color: gray;                                     color: white;
    }                                                           }
    ul.inlinefloat li{                                          ul.inlinefloat li a:hover:not(#active):not(#last){
        float: left;                                                background-color: yellow;
    }                                                           }
    ul.inlinefloat li#last{                                     ul.inlinefloat li:not(#last){
        float: right;                                               border-right: solid white 1px;
    }                                                           }
    ul.inlinefloat li a{
        display: block;
        padding: 10px;
        text-decoration: none;
        font-weight: 700;
        color:black;
    }
