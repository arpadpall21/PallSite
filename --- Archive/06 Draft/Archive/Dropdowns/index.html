Remember This :

- When creating a menu and you display "relative" or "fixed" DO NOT declare "overflow:hidden;" because it will hide all elements whitch goes outside of the menu container and dropdowns and any elements outside of the menu won't be visible!

- When you use ":hover" in CSS only declare the class or id name and not the item type! Exeple ".div:hover" CORRECT - "div.div:hover" INCORRECT

Some Description :

HTML

- Use any element in html to open a dropdown content

- Create a dropdown by using a container element (like "div")

CSS

- The dropdown container is declared hidden and will be displayed on hover

- The dorpdown container is declared position:absolute; to not affect the layout once it appears

Demonstration/Exercise Section :

Dropdown on text

Mouse over this text!

Thada here it is!

CODE 
                -- CODE (HTML) --
                <div class="droptext">
                    <span> Mouse over this text!</span>
                    <div class="drop-content">
                        <p> Thada here it is!</p>
                    </div>
                </div>
    
                -- CODE (CSS) --
                div.droptext{
                    position: relative;
                    display: inline-block;
                }
                div.droptext span{
                    color: red;
                    font-size: 1.5em;
                    font-weight: bold;
                }
                .drop-content{
                    display: none;
                    position: absolute;
                    padding : 10px;
                    box-shadow: 0px 8px 16px 0px rgba(0,0,0,1.0);
                    background-color: red;
                    font-weight: 600;
                    min-width: 200px;
                    text-align: center;
                }
                .droptext:hover .drop-content{
                    display: block;
                }




Dropdown menu

Link 1 Link 2 Link 3
CODE 
                -- CODE (HTML) --
                <div class="dropmenu">
                    <button class="dropbtn"> Dropdown Menu </button>
                    <div class="dropcont">
                        <a href="#"> Link 1 </a>
                        <a href="#"> Link 2 </a>
                        <a href="#"> Link 3 </a>
                    </div>
                </div>
    
                -- CODE (CSS) --
                div.dropmenu{
                    position: relative;
                    display: inline-block;
                }
                .dropbtn{
                    padding: 15px 25px;
                    background-color: darkgreen;
                    color: white;
                    border: none;
                    font-size: 1.2em;
                    cursor: pointer;
                }
                div.dropcont a{
                    display: block;
                    text-decoration: none;
                    padding: 15px;
                    color: white;
                    text-align: center;
                    font-size: 1.2em;
                }
                div.dropcont{
                    display: none;
                    position: absolute;
                    width: 100%;
                    background-color: green;
                }
                div.dropmenu:hover .dropbtn{
                    background-color: green;
                }
                .dropmenu:hover .dropcont{
                    display: block;
                }
                div.dropcont a:hover{
                    background-color: darkgreen;
                }









Image Dropdown

Fancy River

CODE 
                -- CODE (HTML) --
                <div class="img">
                    <img src="../../../../Pictures/river.jpg" alt="river" width="100px">
                    <div class="imgcont">
                        <img src="../../../../Pictures/river.jpg" alt="river" width="500px">
                        <p> Fancy River </p>
                    </div>
                </div>
    
                -- CODE (CSS) --
                div.img{
                    border: solid black 2px;
                    position: relative;
                    display: inline-block;
                    cursor: pointer;
                }
                div.imgcont{
                    display: none;
                    position: absolute;
                    background-color: lightgreen;
                    height: 350px;
                }
                div.imgcont p{
                    text-align: center;
                    line-height: 40%;
                    font-weight: bold;
                }
                div.img:hover div.imgcont{
                    display: block;
                }
