The constants / existsSync() / realpath() / realpath.native() / realpathSync() / realpathSync.native() property and methods 

SYNTAX :    fs.constants                                                               // returns an object containing os specific file system constants  Node.js FS constants [link]
            fs.existsSync(path:str|buf|urlObj)                                         // synchronously checks if the path exists and returns a boolean accordingly 
            
            fs.realpath(path:str|buf|urlObj, {encoding:str}, fn(err, resPath))         // asynchronously resolves the given path (resolves .. and ., symbolic-links are resolved to their original file path) and returns the absolute path in the callback function (UTF-8 convertable paths only) (LINUX realpath())    
            fs.realpathSync(path:str|buf|urlObj, {encoding:str})                       // synchronously resolves the given path (resolves .. and ., symbolic-links are resolved to their original file path) and returns the absolute path (UTF-8 convertable paths only) (LINUX realpath())    
            
            fs.realpath.native(path:str|buf|urlObj, {encoding:str}, fn(err, resPath))  // asynchronously resolves the given path (resolves .. and ., symbolic-links are resolved to their original file path) and returns the absolute path in the callback function (UTF-8 convertable paths only) (LINUX realpath()) (does the same as the realpath() method but faster [TESTED!])    
            fs.realpathSync.native(path:str|buf|urlObj, {encoding:str})                // synchronously resolves the given path (resolves .. and ., symbolic-links are resolved to their original file path) and returns the absolute path (UTF-8 convertable paths only) (LINUX realpath()) (does the same as the realpathSync() method but faster [TESTED!])
DEMO 
    const fs = require('fs');
    
// constants / existsSync() -----------------------------------------------------------
    fs.constants;                                                                      // -> {file system constants...}             // returns an object containing file system constants    
    fs.existsSync('testFile');                                                         // -> true                                   // existing path   
    
// realpath() / realpathSync() --------------------------------------------------------
    fs.realpath('./', function(err, resPath){                                          // asynchronously resolves the absolute path of the current directory   
        if(err) console.log(err);
        resPath;                                                                       // -> 'D:\testFolder'                        // folder we are in 
    });
    
    fs.realpathSync('../');                                                            // -> 'D:\'                                  // synchronously resolves the absolute path of the parent directory    
    
// realpath.native() / realpathSync.native() ------------------------------------------// -! does the same as the above methods but they are faster [TESTED: 2019-10-04]    
    fs.realpath.native('testFile', function(err, resPath){                             // asynchronously resolves the absolute path of the passed file 
        if(err) console.log(err);
        resPath;                                                                       // -> 'D:\testFolder\testFile'               // file's absolute path   
    });
    
    fs.realpathSync.native('symLink_to_testFile');                                     // -> 'D:\testFolder\testFile'               // -! the passed symbolic-link's absolute path is resolved to the original file
